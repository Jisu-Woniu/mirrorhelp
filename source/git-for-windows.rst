=================
Git for Windows
=================

地址
====

https://mirrors.geekpie.club/git-for-windows/

说明
====

Windows版git程序镜像。镜像于git for windows在GitHub的releases。

收录架构
========

x86_64, i386

收录版本
========

所有GA版本。

使用说明
========

下载最新版本中适合自己的版本即可。相关链接中上游提供SHA256校验和。

相关链接
========

:上游: https://github.com/git-for-windows/git/releases
